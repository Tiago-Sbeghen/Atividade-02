Tags Básicas HTML

Neste site possuem informações sobre tags em html

Estrutura Principal:

(html) Essa Tag é a raiz do documento, todas as informções do site devem estar dentro dela
(title) Essa Tag define o título da página no navegador
(meta) Essa Tag define caracteres UTF-8 e configurações de SEO
(head) Essa Tag é como se fosse o cérebro do documento onde são armazenados os meta dados
(body) Essa Tag é como se fosse o corpo do documento onde são armazenados todos os dados que irão aparecer para o usuário

Tags Semânticas:

(h1/h's) As tags h's são utilizadas para demarcar um título. Com ordem de prioridade, sendo o h1 o mais importante e o h6 o de menor relevância na hierarquia
(p) Essa Tag serve para adicionar um parágrafo de texto
(a) Essa Tag serve para criar um hyperlink
(img) Essa Tag serve para adicionar imagens ao site
(div) Essa Tag serve para criar um bloco para layout
(header) Essa Tag serve para definir o cabeçalho
(main) Essa Tag serve para definir o conteúdo principal do site
(footer) Essa Tag serve para definir o rodapé do site

Listas e Tabelas:

(ul)Essa tag serve para criar listas ordenadas sem marcadores
(ol) Essa tag serve para criar listas ordenadas numeradas
(li) Essa tag serve para inserir um item dentro de uma lista

Formatação de elementos:

(strong) Essa Tag serve para deixar o texto em negrito
(em) Essa Tag serve para deixar o texto em itálico
